First Time Setup:
   1. cp config.example.json config.json
   2. nano config.json  # Edit with your values
   3. ./validate-config.sh
   4. ./test-connection.sh
   5. ./batch-configure.sh --sequential

Dependencies:
   Required: jq, curl, bash (>=4.0)
   Optional: Rust toolchain (for Rust CLI)

Production Checklist:
   ✓ Run ./test-all.sh - all tests pass
   ✓ Run ./validate-config.sh - config valid
   ✓ Run ./test-connection.sh - all nodes reachable
   ✓ Verify operator and relayer accounts
   ✓ Review token addresses and exchange rates
   ✓ Backup configuration and keys
   ✓ Test on testnet first!
